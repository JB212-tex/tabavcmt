# tabavcmt
A LaTeX package for generating chemistry reaction progress tables (french "tableaux d'avancement", similar to ICE tables)

# Example output
![Example](images/example.png)

# Local Installation
Copy tabavcmt.sty to your local TeX tree, for example :
## MacOS & Linux
~/texmf/tex/latex/tabavcmt/tabavcmt.sty
## Windows 
<texmf-local>/tex/latex/tabavcmt/tabavcmt.sty

The folder name can be chosen freely.

# Quick test
For a quick test, simply place tabavcm.sty in the same directory as your .tex document.
